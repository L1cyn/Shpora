Что такое Python

Python — это высокоуровневый интерпретируемый язык программирования общего назначения. Он известен своей простотой, читаемостью и очень богатой экосистемой библиотек.

Главная особенность Python — он позволяет быстро переходить от идеи к рабочему коду. Поэтому его используют и для обучения, и для серьёзных промышленных проектов.

Что значит "высокоуровневый"

Это означает, что язык скрывает множество низкоуровневых деталей: управление памятью, сложную работу с ресурсами, служебный синтаксис.

Что значит "интерпретируемый"

Код Python обычно запускается интерпретатором, а не компилируется напрямую в машинный код так, как это происходит в некоторых других языках.

Почему Python любят

  • Его код легко читать.
  • На нём быстро писать полезные программы.
  • Подходит и для маленьких скриптов, и для больших приложений.
  • Есть библиотеки почти для любой задачи.

Рекурсия

Рекурсия — это вызов функцией самой себя. 

def factorial(n):
    if n == 1:
        return 1
    return n * factorial(n - 1)

Что обязательно должно быть

  • Базовый случай — когда рекурсия заканчивается.
  • Рекурсивный шаг — приближение к базовому случаю.

Lambda, map, filter

lambda

square = lambda x: x ** 2
print(square(5))

map()

numbers = [1, 2, 3]
result = list(map(lambda x: x * 2, numbers))

filter()

numbers = [1, 2, 3, 4, 5, 6]
evens = list(filter(lambda x: x % 2 == 0, numbers))

В современном коде нередко проще использовать comprehensions.

Итераторы

Итератор — объект, из которого можно получать элементы по одному. 

numbers = [1, 2, 3]
it = iter(numbers)

print(next(it))
print(next(it))
print(next(it))

Итераторы лежат в основе цикла for.

Генераторы

def count_to(n):
    i = 1
    while i <= n:
        yield i
        i += 1

Что делает yield

Возвращает значение, но сохраняет состояние функции.

Генераторное выражение

squares = (x ** 2 for x in range(5))

Когда полезно

  • Для больших данных.
  • Для экономии памяти.
  • Для потоковой обработки.

Декораторы

def decorator(func):
    def wrapper():
        print("До вызова")
        func()
        print("После вызова")
    return wrapper
@decorator
def hello():
    print("Привет")

hello()

Работа с файлами — одна из самых важных практических тем.

Открытие файла

with open("example.txt", "r", encoding="utf-8") as file:
    content = file.read()

Режимы открытия

РежимЧто делает
rЧтение, файл должен существовать
wЗапись, файл создаётся или очищается
aДобавление в конец, файл создаётся если его нет
xСоздать новый файл, ошибка если уже есть
tТекстовый режим
bБинарный режим
+Чтение и запись одновременно

Изменяемые аргументы по умолчанию

def add_item(item, items=[]):

Лучше:

def add_item(item, items=None):
    if items is None:
        items = []
    items.append(item)
    return items
